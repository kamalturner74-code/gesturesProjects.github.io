Setup: edit the CONFIG block at the top of app.js.

1. Drop your model at model/model.onnx
2. Set LABELS to your class names, in the exact order your model outputs them
3. Add one audio file per label in audio/ and map it in AUDIO_MAP
4. Adjust INPUT_SIZE, LAYOUT, and normalization to match how your model was trained

Serve this folder with a local server (e.g. python3 -m http.server) — opening the file directly (file://) will block the model fetch.
